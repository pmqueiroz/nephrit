<h1 align="center">Nephrite</h1>

**Nephrite** is a Rust-based alternative to Style Dictionary, focused on performance.

Built with **Rust** and exposed to JavaScript via **napi-rs**.

## Development

```bash
pnpm build
pnpm test
```

## Status

Early development. APIs may change.
